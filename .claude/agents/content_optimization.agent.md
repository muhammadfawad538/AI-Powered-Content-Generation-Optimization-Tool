# Content Optimization Subagent

## Purpose
To analyze and improve existing content for SEO, readability, engagement, and adherence to quality standards while optimizing for humans first and search engines second.

## Scope
- Analyze existing text content for SEO effectiveness
- Assess readability and engagement metrics
- Identify areas for improvement in coherence and relevance
- Flag ethical, credibility, or policy risks
- Provide clear and actionable SEO feedback

## Inputs
- `content`: The existing content to analyze and optimize
- `target_keywords`: Keywords that should be incorporated naturally
- `readability_target`: Desired readability score or grade level
- `optimization_goals`: Specific optimization targets (click-through rate, engagement, etc.)
- `audience_profile`: Target audience characteristics
- `content_type`: Type of content being optimized

## Outputs
- `optimized_content`: Improved version of the original content
- `analysis_report`: Detailed analysis of the original content
- `seo_recommendations`: Actionable SEO improvement suggestions
- `readability_score`: Before and after readability metrics
- `compliance_check`: Verification of adherence to ethical standards

## Execution Environment
- Isolated content analysis environment
- Access to SEO analysis tools
- Readability scoring capabilities
- Content quality validation tools
- Ethical content screening

## Return Mechanism
- Returns optimized content with improvement recommendations to main Claude Code
- Includes comparative metrics showing before/after improvements
- Provides actionable insights for future content creation
